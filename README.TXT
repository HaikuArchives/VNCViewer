There is one program here:

	VNCviewer - this is the VNC viewer, or client, program for BeOS.

The program was built using Metrowerks Codewarrior 1.6 for BeOS/x86
and contains a project file suitable for building it.

The original VNC source code from the ORL can be obtained from the
ORL VNC homepage at http://www.orl.co.uk/vnc/

The latest versions of the VNCviewer for BeOS can be obtained from
http://www.abstrakt.ch/be/
